# Trendova Web

Landing page لـ [Trendova](https://trendova.ai) — وكيل AI لأعمالك.

## تشغيل محلي

```bash
npm install
npm run dev
```

الموقع يشتغل على http://localhost:3000

## بناء للإنتاج

```bash
npm run build
npm start
```

## Deploy على Railway

المشروع جاهز للنشر على Railway عبر Dockerfile مع standalone output.

1. ارفع الكود على GitHub
2. أنشئ مشروع جديد على Railway وربطه بالـ repo
3. Railway سيكتشف `railway.json` ويبني تلقائياً
